# Altrodav Data Analyst Task

This project is created for the Altrodav Technologies SQL assessment.

In this project, I solved 5 business problems using PostgreSQL queries and generated analytical insights from the data.

## Files Included

- queries.sql → Contains all SQL queries
- INSIGHTS.md → Contains business insights and recommendations
- results/ → Contains CSV output files for all queries

## Topics Used

- SQL Joins
- Aggregations
- Window Functions
- CTEs
- PostgreSQL Queries
